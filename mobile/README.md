# Pédja Mobile App

Application mobile React Native avec Expo pour la plateforme éducative Pédja.

## Installation

```bash
npm install
```

## Démarrage

```bash
# Démarrer le serveur Expo
npm start

# Lancer sur Android
npm run android

# Lancer sur iOS
npm run ios
```

## Structure

- `/app` - Routes et écrans (Expo Router)
- `/components` - Composants réutilisables
- `/lib` - Utilitaires et helpers
- `/assets` - Images et ressources

## Features actuelles

- ✅ Navigation avec Expo Router
- ✅ Écrans d'authentification (Login/Register)
- ✅ Navigation par onglets (Dashboard, Examens, Profil)
- ⏳ Intégration API backend
- ⏳ Stockage sécurisé des tokens
